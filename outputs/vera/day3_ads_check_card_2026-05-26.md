## Facebook Ads — What to Check

Open Facebook Ads Manager → All Campaigns

| Metric | Green (do nothing) | Yellow (watch it) | Red (act now) |
|--------|-------------------|-------------------|---------------|
| **Spend** | On track ($30/day ±15%) | Underspending ($15–$25) | $0 spent by noon |
| **CPM** (cost per 1,000 impressions) | $10–$25 | $25–$40 | >$40 (audience too narrow) |
| **CTR** (click-through rate) | >1.0% | 0.5–1.0% | <0.5% |
| **Leads** | Any lead at all | 0 leads, spending | 0 leads + $0 spend |

### If CTR is below 0.5%:
→ Only then — swap the hook text using `outputs/rick/facebook_ad_creative_voc_refresh_2026-05-24.md`. Use the "green and slimy" or "roof looks black" variant. ONE change, then leave it for 4 more days.

### If $0 spend:
→ Check billing → check audience size (must be >100,000) → check ad approval status. Do NOT duplicate the campaign.

### The Big Rule:
**DO NOT turn ads off because 0 leads on Days 1-2.** Facebook's algorithm needs 50 optimization events before it knows who to show ads to. Day 1-2 is data gathering. Day 3 is the first real signal.
